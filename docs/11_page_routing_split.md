# Step 11. 기능을 페이지로 나누기

> 목적: Step 10에서 모은 **기능을 화면(페이지) 단위로 나눠** 배치한다.

> **Guide:**  
> 1. Agent가 `docs/10_feature_list.md`의 기능 목록을 읽고 **페이지 구성 초안**을 작성한다.  
> 2. 각 페이지에 **어떤 기능(F-xx)** 이 들어가는지 나열한다.  
> 3. 팀이 페이지 수·이름·구성을 보며 수정한다.  
> - Step 10의 기능은 **어느 페이지에든 모두 배치**한다. (이 단계에서 기능을 빼지 않는다)  
> - 사용자 흐름이 자연스럽게 이어지도록 페이지를 묶는다. (예: 로그인 → 메인 → 상세)  
> - 한 페이지에 기능이 너무 많으면 **페이지를 나눈다.**  
> - **우선순위·스펙 아웃(이번에 만들지 않을 기능)** 은 Step 14 `docs/14_priority_matrix.md`에서 정한다.

**프롬프트 예시**

```
docs/10_feature_list.md와 docs/04~09 맥락을 읽고 docs/11_page_routing_split.md를 채워줘.

규칙:
- Step 10의 모든 기능(F-xx)을 어느 페이지에든 배치 (누락 없음)
- 페이지마다 이름·경로·목적·포함 기능 목록을 작성
- 기능 ID(F-xx)와 기능명을 함께 적기
- 해커톤 MVP 기준으로 페이지 수는 과하지 않게 (보통 3~6개)
- "페이지 간 이동 흐름" 섹션도 채우기
- 구조와 상단 Guide는 유지
```

> 참고: 단일 페이지 Canvas 게임이라 "페이지"는 라우트가 아니라 **하나의 상태 머신(state machine)** 으로 전환되는 화면이다. 모든 것이 `/` 한 경로 위에서 `MENU → PLAYING → RESULT` 상태로 바뀐다.

## 페이지 1: 메인 화면 (MENU)

> **상태:** `MENU`  
> **목적:** 접속 즉시 뜨는 가벼운 시작 화면. 타이틀·규칙 한 줄을 보여주고 버튼 한 번으로 바로 플레이로 넘긴다. (인트로·로딩 없이 화면 자체는 즉시 표시 → 즉시성 유지)

- **F-01** 즉시 진입 (로딩·인트로 없이 메인 화면이 바로 뜸)
- **F-25** 조작·규칙 힌트 한 줄 표시
- **F-30** 최고 점수/도달 웨이브 표시 (있으면)
- 게임 시작 버튼 → `PLAYING`으로 전환

## 페이지 2: 게임 화면 (PLAYING)

> **상태:** `PLAYING`  
> **목적:** 실제 플레이가 벌어지는 핵심 화면. 맵·타워·적·전투 + HUD·타워 선택 바·웨이브 조작이 함께 있다.

**플레이 보드**

- **F-02** 격자 기반 게임 맵 렌더링
- **F-03** 적 경로 표시
- **F-04** 설치 가능한 빈 타일 표시
- **F-05** 빈 타일 클릭으로 타워 설치
- **F-07** 타워 사거리 미리보기 원
- **F-08** 호버 타일 설치 가능/불가 하이라이트
- **F-09** 여러 종류의 타워
- **F-10** 타워 자동 조준·발사
- **F-11** 발사체 이동·명중 처리
- **F-12** 적 스폰·경로 이동
- **F-13** 여러 종류의 적
- **F-14** 적 HP·체력바 표시
- **F-15** 적 피해 적용·처치
- **F-17** 웨이브별 적 구성·난이도 변화

**HUD / 컨트롤 바 (보드 위 상시 표시)**

- **F-06** 타워 종류 선택 UI (이름·비용·특성)
- **F-16** 웨이브 시작 버튼·순차 스폰
- **F-18** 골드 시스템 (처치 시 획득)
- **F-19** 골드로 설치·강화 비용 지불
- **F-20** 웨이브 클리어 보너스 골드
- **F-21** 타워 강화(업그레이드)
- **F-22** 타워 판매(환불)
- **F-23** 목숨 시스템 (적 도달 시 감소)
- **F-24** 상단 HUD (골드·목숨·웨이브)

## 페이지 3: 결과 화면 (RESULT)

> **상태:** `RESULT`  
> **목적:** 목숨 0(게임오버) 또는 전 웨이브 방어(승리) 시 뜨는 화면. 성패·성취를 명확히 보여주고 재도전·공유로 연결.

- **F-26** 게임오버 처리 (목숨 0)
- **F-27** 승리 처리 (전 웨이브 방어)
- **F-28** 결과 요약 (도달 웨이브·처치 수·남은 목숨)
- **F-29** 다시 하기 버튼 → `PLAYING`(또는 `MENU`)으로 전환
- **F-30** 점수/도달 웨이브 기록 (최고기록 갱신)
- **F-31** 결과 링크 공유

## 페이지 간 이동 흐름

> 사용자가 페이지를 어떻게 오가는지 순서대로 적는다.

1. 링크 접속 → **메인 화면(MENU)** 이 즉시 뜨고, 규칙 한 줄·최고기록이 보인다.
2. **[게임 시작]** 클릭 → **게임 화면(PLAYING)** 으로 전환.
3. 게임 화면에서 타워를 골라 빈 타일에 설치하고 "웨이브 시작"을 누른다.
4. 적을 방어하며 골드를 벌고, HUD로 타워를 추가·강화·판매한다.
5. 웨이브를 클리어하면 3~4를 반복하며 다음 웨이브를 진행한다.
6. 목숨이 0이거나 전 웨이브를 막으면 **결과 화면(RESULT)** 으로 전환.
7. 결과 화면에서 **[다시 하기]** 로 게임을 재시작하거나, 결과를 공유한다.
