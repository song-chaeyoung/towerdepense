# Step 06. PainPoint / WowPoint

> 목적: `docs/05` 기존 여정에서 Pain·Wow를 추출하고, 우리 서비스의 개선·차별 방향으로 정리한다.

> **Guide:** 기존 여정에서 사용자가 **불편했던(Pain)** 점은 우리 서비스에서 개선하고, **좋았던(Wow)** 점은 가져와 매력 있고 가치 있는 서비스를 만든다. `docs/05`의 모든 여정·단계를 검토해 Pain/Wow를 빠짐없이 나열한다.

**프롬프트 예시**

```
docs/05_existing_customer_journey.md를 읽고 docs/06_pain_wow_points.md를 채워줘.
각 여정·단계에서 사용자가 불편했던 Pain, 좋았던 Wow를 모두 추출해 나열해줘.
하나로 합치거나 우선순위를 정하지 말고, 발견한 항목을 빠짐없이 bullet로 정리해줘.
구조와 상단 Guide는 유지하고, 본문만 채워줘.
```

## PainPoint

- 잠깐 하려는데 앱 설치·수백 MB 다운로드가 부담스럽다. (여정1 설치)
- 로그인·약관 동의·긴 튜토리얼 때문에 플레이까지 오래 걸린다. (여정1 실행, 여정3 진입)
- 광고·과금 유도가 게임을 자주 끊어 몰입이 깨진다. (여정1 플레이, 여정2 방해)
- 게임 사이트가 팝업·배너로 지저분하고 진짜 게임 링크를 찾기 어렵다. (여정2 검색)
- 로딩이 길고 중간에 딴 페이지로 튄다. (여정2 로딩)
- 공유받은 링크가 가입·설치를 요구해 결국 포기하게 된다. (여정3 포기)
- 다시 그 게임을 찾아 들어가기가 귀찮다. (여정2 종료)

## WowPoint

- 조작(마우스로 타워 배치)이 이미 익숙해 금방 적응된다. (여정2 플레이)
- 링크가 웹에서 바로 실행되면 부담 없이 한 판 하게 된다. (여정3 대안)
- 링크만 보내면 되니 친구에게 퍼뜨리기 쉽다. (여정3 공유)
- 규칙이 익숙한 타워디펜스라 설명 없이도 재미를 안다. (여정1 플레이)
