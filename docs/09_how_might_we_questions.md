# Step 09. "어떻게 하면 ~" 질문 만들기

> 목적: 서비스 방향을 맞춘 뒤, **더 구체적인 생각**으로 수렴한다.

> **Guide:**  
> 1. Agent가 `docs/04`, `docs/06`, `docs/08`을 읽고 **"어떻게 하면 ~ 할 수 있을까?"** 형태의 질문을 만든다. 직접 질문을 추가해도 좋다.
> 2. 팀이 질문을 보며 답을 이야기한다.
> 3. 각 질문 아래 **답**을 적는다. (타이머 ON)  
> - 질문은 **많을수록** 좋고, 하고 싶은 말이 많은 질문이 좋다.  
> - **실제로 어떻게 할지** 떠올릴 수 있는 질문을 한다. (예: 사용자를 어떻게 모을까? → 너무 넓음)  
> - 답은 **질문과 같은 말 반복·뭉뚱그린 표현 금지**. 구체적인 문장으로 쓴다.  
>   - (X) 어떻게 하면 사용자가 이 기능을 더 잘 쓸까? → 사용자들이 더 쉽게 쓰게 만들면 될 것 같아요  
> - 참고한 서비스·자료가 있으면 답에 적는다.

**프롬프트 예시**

```
docs/04_service_goal_target_value.md, docs/06_pain_wow_points.md, docs/08_word_cloud.md를 읽고
docs/09_how_might_we_questions.md의 "Agent가 만든 질문" 섹션만 채워줘.

질문 규칙:
- "어떻게 하면 ~ 할 수 있을까?" 형태
- docs/04 목적·핵심 가치, docs/06 불편한 점·좋았던 점, docs/08 키워드를 골고루 반영
- 실제 방법·아이디어가 떠오를 수 있는 질문 (너무 넓은 "사람 모으기"류 X)
- 최소 10개 이상, 서로 겹치지 않게
- 각 질문 아래 답 칸은 비워둬 (팀이 채움)
- "팀이 추가한 질문" 섹션은 건드리지 마
- 구조와 상단 Guide는 유지
```

## Agent가 만든 질문

> `docs/04`(목적·가치), `docs/06`(불편한 점·좋았던 점), `docs/08`(키워드)을 바탕으로 Agent가 채운다.

### 질문 1: 어떻게 하면 링크를 연 사용자가 설치·로그인 없이 곧바로 플레이하게 할 수 있을까?

- 접속하면 로딩·인트로 없이 게임 맵과 "웨이브 시작" 버튼이 바로 뜨게 한다.
- 서버·계정 없이 프론트(Canvas)만으로 완결시켜 첫 화면 = 플레이 화면으로 만든다.

### 질문 2: 어떻게 하면 튜토리얼 없이도 규칙(짓고·막고·강화)을 이해하게 할 수 있을까?

- 설치 가능한 빈 타일과 적 경로를 색·모양으로 뚜렷이 구분해 보여준다.
- 골드/목숨/웨이브를 화면 상단에 항상 표시하고, 첫 진입 시 짧은 힌트 한 줄을 띄운다.

### 질문 3: 어떻게 하면 설치 가능한 자리와 적이 지나가는 경로를 한눈에 구분하게 할까?

- 경로 타일은 도로색, 설치 가능 타일은 잔디색으로 칠하고 격자선을 그린다.
- 마우스를 올린 타일에 설치 가능/불가를 미리 하이라이트로 표시한다.

### 질문 4: 어떻게 하면 타워를 클릭 전에 종류·비용·효과를 알고 고르게 할까?

- 타워 선택 버튼에 이름·비용·간단한 특성(사거리/피해)을 함께 적는다.
- 타워 선택 후 커서 위치에 사거리 원(range)을 미리 보여준다.

### 질문 5: 어떻게 하면 매 판 다른 전략이 나오도록 리플레이성을 줄까?

- 사거리/연사/광역/둔화 등 성격이 다른 타워를 여러 종류 제공한다.
- 웨이브마다 적 구성(빠른 적·탱커 등)을 달리해 배치 조합을 고민하게 한다.

### 질문 6: 어떻게 하면 골드 관리로 긴장감 있는 선택을 만들까?

- 적을 잡으면 골드를 주고, 타워 설치·강화에 골드를 쓰게 해 자원 압박을 만든다.
- 웨이브를 넘길 때 보너스 골드를 줘서 "지금 강화 vs 아껴서 새 타워" 고민을 남긴다.

### 질문 7: 어떻게 하면 한 판의 성패와 성취를 명확히 남길까?

- 목숨 0이면 게임오버, 전 웨이브 방어 시 승리 화면을 확실히 보여준다.
- 도달 웨이브·처치 수·남은 목숨 등 결과 지표를 마지막에 요약해 준다.

### 질문 8: 어떻게 하면 재밌게 끝낸 사용자가 다시·친구와 하게 할까?

- 결과 화면에 "다시 하기" 버튼을 두어 즉시 재도전하게 한다.
- 점수/도달 웨이브를 결과에 남겨 이후 링크 공유·하이스코어로 확장할 여지를 둔다.
