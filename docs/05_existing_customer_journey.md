# Step 05. 기존 고객 여정

> 목적: `docs/04`에서 정의한 서비스 목적을, **지금** 사용자가 어떤 방식으로 달성하는지 정리한다.

> **Guide:** **우리가 만들 서비스의 여정이 아니다.** (우리 서비스 여정은 Step 12 `docs/12_customer_journey_map.md`) 유사 서비스·대체제·우회 방식마다 **여정을 하나씩** 두고, 단계별 사용자 행동·생각/감정을 표로 정리한다. 각 단계는 상세하게 작성한다. 이 내용을 토대로 Step 06에서 Pain/Wow 인사이트를 추출한다.

## 여정 1

> 모바일 앱스토어에서 타워디펜스 게임을 설치해서 즐긴다

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 검색 | 앱스토어에서 "tower defense" 검색, 목록 훑어봄 | 뭐가 재밌는지 모르겠고 비슷비슷해 보임 |
| 설치 | 수백 MB짜리 앱을 다운로드·설치 후 대기 | 잠깐 하려던 건데 용량·시간이 부담스럽다 |
| 실행 | 로그인/약관 동의, 긴 인트로·튜토리얼 진행 | 빨리 게임하고 싶은데 왜 이렇게 오래 걸리지 |
| 플레이 | 첫 판을 시작, 조작에 적응 | 재밌긴 한데 광고·과금 유도가 자주 끊는다 |
| 이탈 | 며칠 뒤 잘 안 켜게 되고 앱 삭제 | 굳이 앱까지 깔 일은 아니었던 것 같다 |

## 여정 2

> 웹 브라우저에서 무료 게임 사이트의 타워디펜스를 즐긴다

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 검색 | 포털에 "타워디펜스 게임" 검색해 사이트 진입 | 광고·팝업이 많아 진짜 게임 링크 찾기 어렵다 |
| 로딩 | 배너·플러그인·긴 로딩을 지나 게임 진입 | 로딩이 길고 중간에 딴 데로 튄다 |
| 플레이 | 게임 시작, 마우스로 타워 배치 | 조작은 익숙한데 화면이 지저분하다 |
| 방해 | 중간중간 전면 광고가 게임을 끊음 | 몰입이 자꾸 깨져서 답답하다 |
| 종료 | 한 판 끝나고 탭을 닫음 | 다시 그 사이트를 찾아 들어가긴 귀찮다 |

## 여정 3

> 친구가 공유한 게임 링크를 받아서 즐긴다

| 단계 | 사용자 행동 | 생각/감정 |
|---|---|---|
| 수신 | 메신저로 게임 링크를 받음 | 뭔지 궁금해서 눌러본다 |
| 진입 | 링크를 열자 로그인·설치를 요구함 | 그냥 잠깐 보려 했는데 절차가 걸린다 |
| 포기 | 가입이 귀찮아 뒤로가기로 나감 | 바로 플레이됐으면 했을 텐데 아쉽다 |
| 대안 | 링크가 웹에서 바로 실행되면 한 판 함 | 이런 건 부담 없어서 좋다 |
| 공유 | 재밌으면 다른 친구에게 다시 공유 | 링크만 보내면 되니 퍼뜨리기 쉽다 |
